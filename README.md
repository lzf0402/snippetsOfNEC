##  轻松几步，制作你自己的代码片段  ##

1. Sublime text ——> tools ——> New Snippet

新建一个Snippet，保存成`.sublime-snippet`后缀的文件，Sublime Text默认会保存到`C:\Users\yourname\AppData\Roaming\Sublime Text 2\Packages\User`此目录下。当然你也可以在此目录下新建文件夹，把代码片段进行分类。
